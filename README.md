# Odin-etch-a-sketch
A simple etch-a-sketch style tool for making highly pixelated images

Constructs a 16\*16 grid of cells using JavaScript which can be coloured incrementally by hovering the mouse over them.
Provides two options for colouring: pure black and white, or randomised colours, allowing the user to switch between them through the UI.
Also allows the user to reset the grid with a specified grid size, allowing any n\*n grid of size n=1 to n=100.

Useable at GitHub Pages link: https://milesrigby.github.io/Odin-etch-a-sketch/index.html
